## 📌 Über dieses Projekt

**Play a Game** ist eine Sammlung verschiedener kleiner Spiele, die teilweise in **JavaScript** oder mit der **Godot Engine 4.6** entwickelt wurden.

Ziel dieses Projekts ist es, eine übersichtliche Plattform für experimentelle und klassische Mini-Games bereitzustellen.
